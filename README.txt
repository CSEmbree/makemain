// COMPILING
To compile makemain, execute the following:

    gcc mm.c -o mm


// RUNNING
To run makemain, execute a line with the format:

	./mm <fileName>.<extension>


//SUPPORT
Supported languages formats:
C
	./mm hello.c

C++/CPP
	./mm hello.c++
	OR
	./mm hello.cpp

PYTHON
	./mm hello.py

JAVA
	./mm hello.java
